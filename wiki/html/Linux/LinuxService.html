nsswitch - Name Service Switch

[http://www.cnblogs.com/cute/archive/2012/05/17/2506342.html 详解]

e.g #

  1. /etc/nsswitch.conf (originally from netkit_base-0.17)

#

  1. An example Name Service Switch config file. This file should be
  2. sorted with the most-used services at the beginning.

#

  1. The entry '[NOTFOUND=return]' means that the search for an
  2. entry should stop if the search in the previous entry turned
  3. up nothing. Note that if the search failed due to some other reason
  4. (like no NIS server responding) then the search continues with the
  5. next entry.

#

  1. Legal entries are:

#

  1. nisplus or nis+ Use NIS+ (NIS version 3)
  2. nis or yp Use NIS (NIS version 2), also called YP
  3. dns Use DNS (Domain Name Service)
  4. files Use the local files
  5. db Use the local database (.db) files
  6. compat Use NIS on compat mode
  7. [NOTFOUND=return] Stop searching if not found so far

# passwd: files nisplus nis shadow: files nisplus nis group: files nisplus nis hosts: files nisplus nis dns services: files nisplus networks: files protocols: files nisplus rpc: files ethers: files netmasks: files netgroup: files nisplus publickey: nisplus automount: files nisplus aliases: files nisplus bootparams: nisplus [NOTFOUND=return] files
