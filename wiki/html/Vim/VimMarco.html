run the marco in register

@a #执行a中存储的指令 @@ #重复执行

execute the macro recorded in register a on all lines of the current file. :%normal @a

run the marco to the selected virtual block, with a visually selected set of lines: :normal @a

:reg : 列出寄存器, 显示所有当前的register "ap #把register a中的内容贴到当前位置 .. #现在你可以修改它了 "add#删除之，重新存入register a
